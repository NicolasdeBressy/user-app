# Description

Cette application récupère des données utilisateur à partir de l'API randomuser.me et les affiche sous forme de cartes stylisées.

## Fonctionnalités

- Affichage des données d'utilisateurs, y compris leur nom, leur photo, leur ville et leur ancienneté en tant que membre.
- Les données sont mises à jour automatiquement à chaque chargement de la page.
- Les cartes sont stylisées pour une présentation attrayante.

## Technologies Utilisées

- HTML
- CSS
- JavaScript

## Comment Utiliser

1. Clonez ce dépôt sur votre machine locale.
2. Ouvrez le fichier `index.html` dans votre navigateur web.
3. Les cartes d'utilisateurs seront affichées automatiquement.
